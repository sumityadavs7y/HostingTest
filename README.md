# Hello World Node.js Application

A simple Node.js HTTP server that responds with "Hello World!".

## Installation

No dependencies required! This application uses only Node.js built-in modules.

## Running the Application

Start the server:

```bash
npm start
```

Or run directly:

```bash
node index.js
```

The server will start on port 3000. You can access it at:
- http://localhost:3000

## What it does

- Creates an HTTP server using Node.js built-in `http` module
- Listens on port 3000
- Responds to all requests with "Hello World!"
